# Quora-like CRUD Posts App

A simple post sharing app built with Node.js, Express, and EJS.

## Features
- Add new posts
- View all posts
- Edit and delete posts
- Styled with custom CSS

## Tech Stack
- Node.js
- Express.js
- EJS
- UUID
- Method-Override

## How to Run
```bash
npm install
node index.js


Visit http://localhost:3000/posts
